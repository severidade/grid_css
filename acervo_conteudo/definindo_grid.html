Definindo o grid

Criando o Conteiner

A primeria coisa a fazer é criar um grid container com as declarações CSS display: grid ou display: inline-grid. Assim declarado, todos os elementos filhos diretos daquele container se transformam em grid items.

É preciso então estabelecer explicitamente o grid. No Exemplo 01 temos um grid de 3 colunas com o valor 1fr e duas linhas como o valor de 200px.

A unidade fr representa uma fração do espaço disponível no container do grid.

Com grid-gap: 20px é definida a separaração tanto em colunas quanto em linhas dos grid-itens no valor de 10px.

Exemplo 01
				
.cards { 
	display: grid;
	grid-template-columns: 1fr 1fr 1fr;
	grid-template-rows: 100px 100px;
	grid-gap: 10px;
}

Proporção entre colunas

A proporção entre as colunas pode ser variável. No Exemplo 2 foram ciadas três colunas mas o espaço do grid foi dividido em 4 frações. As duas primeiras colunas ocupam respectivamente uma fração e a terceira coluna duas frações.

Exemplo 02
				
.cards { 
	display: grid;
	grid-template-columns: 1fr 1fr 2fr;
	grid-template-rows: 100px 100px;
	grid-gap: 10px;
}

Unidades de comprimento

É possível misturar a unidade fr com outras unidades de comprimento como: px, vw, em. Veja o Exemplo 04 o conteiner apresenta três colunas a primeira com o valor fixo de 30px. O espaço restante é dividido em três frações sendo que a segunda coluna representa uma fração 1fr e a terceira representa duas frações 2fr.

Exemplo 03
				
.cards { 
	display: grid;
	grid-template-columns: 30px 1f 2fr;
	grid-template-rows: 100px 100px;
	grid-gap: 10px;
}

Otimizando a escrita

A grafia reduzida grid-template-columns: repeat(2, 2fr) 1fr;) pode para otimizar a escrita evitando a repetição. No Exemplo 04 o conteiner foi dividido em 5 partes. As duas primeiras colunas representam 2 partes cada sendo que a última representa uma parte.

Exemplo 04
				
.cards { 
	display: grid;
	grid-template-columns: repeat(2, 2fr) 1fr;);
	grid-template-rows: 100px 100px;
	grid-gap: 10px;
}

Grid implicito

Nos exempos mostrados até o momento foram definidos as colunas e as linhas do grid. Todavia se as linhas não forem definidas elas serão automaticamente criadas. É o caso do Exemplo 05

As colunas precisam ser declaradas explicitamente.

Exemplo 05
				
.cards { 
	display: grid;
	grid-template-columns: repeat(3, 1fr);
	grid-gap: 10px;
}

Para definir a alutra da linha declarada implicitamente basta usar propiedade grid-auto-rows atribuindo a ela um valor como no Exemplo 6.

	grid-template-columns: repeat(3, 1fr);
	grid-auto-rows: 100px;
	grid-gap: 10px;
