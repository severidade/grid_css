Grid Implícito

Adicionando linhas implicitamente

Nos exempos mostrados até o momento foram definidos as colunas e as linhas do grid. Todavia se as linhas não forem definidas elas serão automaticamente criadas. É o caso do Exemplo 05

As colunas precisam ser declaradas explicitamente.

Exemplo 05
				
.cards { 
	display: grid;
	grid-template-columns: repeat(3, 1fr);
	grid-gap: 10px;
}

Linhas implícitas com altura fixa

Para definir a alutra da linha declarada implicitamente basta usar propiedade grid-auto-rows atribuindo a ela um valor como no Exemplo 6.

Exemplo 06
				
.cards { 
	display: grid;
	grid-template-columns: repeat(3, 1fr);
	grid-auto-rows: 100px;
	grid-gap: 10px;
}

Adicionando colunas automaticamente

Para estabelecer o ajuste de colunas no grid automaticamente basta usar o valor auto-fill definindo uma largura para a coluna. Caso o grid-item não caiba naquela linha ele é automaticamente enviado para a linha abaixo.

Haverá espaços vazios no grid-conteiner caso:

  • o valor seja menor que o especificado para a largura da coluna;
  • Se a largura total de todos os grid-itens mais os grid-gaps for menor que o comprimento do conteiner.

Redimensione a caixa cinza do Exemplo 7 e veja o resultado

Exemplo 07
				
.cards { 
	display: grid;
	grid-template-columns: repeat(auto-fill, 100px);
	grid-auto-rows: 100px;
	grid-gap: 10px;
}

A função minmax( )

Com a função minmax é possível criar um grid mais flexivel diminuindo a possibilidade de aparecimento de espaços vazios pois largura da coluna é definida por um intervalo minmax(200px, 2fr)

Exemplo 08
				
.cards { 
	display: grid;
	grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
	grid-auto-rows: 100px;
	grid-gap: 10px;
}
